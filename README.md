# NGI – Property Hub

  Internal real estate investment app for Nexus Growth Inc.

  Built with Expo (React Native) for iOS. Manages fix-and-flip projects and rental properties in New Jersey.

  ## Stack
  - Expo SDK 54 / React Native / TypeScript
  - expo-router v6 (file-based routing)
  - AsyncStorage (local data persistence)
  - Inter font, react-native-gesture-handler
